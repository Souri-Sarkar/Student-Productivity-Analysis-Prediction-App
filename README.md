# 📊 Student Productivity Analysis & Prediction App

This project predicts student productivity based on daily habits using Machine Learning and provides a Flutter-based user interface.

---

## 🚀 Tech Stack
- Python
- FastAPI
- Scikit-learn
- Flutter
- HTML/CSS (API testing via Swagger)

---

## 📌 Features
- Predict student productivity score
- Input: study hours, phone usage, social media, gaming, stress, etc.
- Real-time API using FastAPI
- Flutter UI for user interaction

---

## 🧠 Machine Learning Model
- Trained using student behavior dataset
- Saved using `pickle (model.pkl)`
- Predicts productivity based on input features

---

## ⚙️ How to Run

### Backend (FastAPI)
- uvicorn app:app --reload

### Frontend (Flutter)
- C:\flutter\bin\flutter.bat run -d web-server
